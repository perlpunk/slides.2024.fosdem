Use the right tools!

  • If you are unsure, quote it. It doesn't hurt.
  • Use a linter, e.g. yamllint, that tells you you have unnecessary quotes
  • Use a tool that automatically removes useless quotes before committing, e.g. yamltidy
  • You are using formatters for other languages as well, right?
