Github Workflow

jobs: example_matrix: strategy: matrix: version: [10, 12, 14] steps: - uses: actions/setup-node@v3 with: node-version: ${{ matrix.version }}
  • The ${{ ... }} is a bit nicer than Jinja as one usually does not need quotes
  • Many people think ${{ ... }} is a part of YAML
  • Intro on their website: "Workflow files use YAML syntax, and must have either a .yml or .yaml file extension. If you're new to YAML and want to learn more, see 'Learn YAML in Y minutes.'"
